# EllipticSurface_ZetaFunctions
This is code, written by Blair Butler, designed to compute the Zeta Function of K3 Elliptic Surfaces over
Finite Fields of prime characteristic.
Eventually I hope to be able to compute the discriminat of the Neron-Severi group (mod squares), in order to determine
an upper bound on the Picard Number of a K3 Elliptic Surface over CC.
